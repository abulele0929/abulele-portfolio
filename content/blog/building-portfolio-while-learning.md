Title: Building This Portfolio While Learning
Date: 2026-06-24
Category: building
Tags: pelican, static site, portfolio, learning
Summary: I'm building my portfolio using Pelican and learning about static site generation in the process.

This very site is a project in itself. I started with a Flask app, but after some research I realised a static site generator (Pelican) is a much better fit for a portfolio: no server, no costs, and everything is just files.

I'm using vanilla CSS and JavaScript, no frameworks. The blog posts are Markdown. The whole thing builds into a folder of HTML that I can drag onto Netlify. It's simple, fast, and perfectly suited to where I am right now: a beginner who wants to ship.

I'll document the whole build process here. If you're thinking of making your own portfolio, hopefully my notes help.
